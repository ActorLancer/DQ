# V1 非目标清单（CTX-018）

## 1. 明确不做（V1 非承诺）

- 生产级公链锚定作为主交易依赖。
- 凭证/NFT 正式规模化发放与运营体系。
- 跨境自动交付与跨司法辖区自动放行。
- 面向自然人的大众供给市场（C2C）。
- `V2/V3` 隐私计算正式能力（`C2D / FL / MPC / TEE / ZKP`）。

## 2. 仅允许边界预留

- 公链展示能力可做可关闭演示开关，不得进入 V1 主闭环关键路径。
- 隐私计算相关接口只允许预留扩展点，不得在 V1 落地正式交易链路。
- 复杂跨平台连接器框架只保留命名与接口预留，不做完整实现。

## 3. 判定规则

- 若某实现会改变 V1 主对象状态机或交易闭环依赖链，即视为超范围。
- 若某能力缺少 V1 冻结任务编号支撑，即不得纳入当前批次实现。
