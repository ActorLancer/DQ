# V1 退出标准冻结（CTX-015）

## 1. 退出门槛（必须同时满足）

1. 五条标准链路端到端验证通过。
2. 八个标准 SKU 主链路可独立跑通且可审计联查。
3. 生命周期对象（`Order / DigitalContract / Authorization / Delivery / Settlement / Dispute / BillingEvent`）均可独立落库并可追踪状态推进。
4. 证据包可导出并完成至少一次模拟监管检查。

## 2. 关键质量指标门槛（V1）

- 订单主链路成功率达到项目验收门槛（以冻结测试矩阵执行结果为准）。
- 验收成功率达到项目验收门槛（含拒收/争议分支可追溯）。
- 账单准确率达到项目验收门槛（含订阅与按量场景）。

说明：具体数值门槛由后续 TEST 验收批次按冻结测试报告固化，不在本任务中臆造。

## 3. 必须通过的能力面

1. 支付回调幂等与异常分支。
2. outbox / dead-letter / replay 一致性链路。
3. 搜索/推荐结果回主库放行校验。
4. 高风险动作 step-up、SoD 与审计留痕。
5. 链确认回写只更新镜像状态，不篡改业务主状态。

## 4. 不达标即不得宣告 V1 完成

- 仅主流程通过、异常分支缺失。
- 接口/事件/权限/审计任一侧存在未收口缺口。
- 无法提供可复核证据链与监管检查样例。
