# 统一错误码映射规范（BOOT-012）

本文件定义工程内错误码分组与命名前缀，具体业务码以 `开发准备/统一错误码字典正式版.md` 为准。

## 前缀分组

- `IAM_`：身份、会话、认证与授权前置错误
- `CAT_`：目录、商品、SKU、模板与元信息错误
- `TRD_`：订单、合同、生命周期主状态错误
- `DLV_`：交付、下载票据、共享、查询执行错误
- `BIL_`：计费、支付、结算、退款相关错误
- `AUD_`：审计、证据、回放、法务保全错误
- `OPS_`：运维、一致性修复、搜索重建、死信处理错误

## 命名规则

- 采用 `PREFIX_SCENARIO_RESULT` 结构。
- 保持语义稳定，不在不同模块重复定义同一语义不同编码。
- 接口响应优先返回稳定 `code`，`message` 可本地化。
