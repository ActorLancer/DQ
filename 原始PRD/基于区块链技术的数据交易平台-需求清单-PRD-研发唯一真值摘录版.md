## 2.1 冻结摘要表

### 2.1.1 V1 正式权利枚举

- `use`
- `access`
- `query`
- `result_get`
- `internal_share_limited`

### 2.1.2 V1 标准 SKU

- `FILE_STD`
- `FILE_SUB`
- `SHARE_RO`
- `API_SUB`
- `API_PPU`
- `QRY_LITE`
- `SBX_STD`
- `RPT_STD`

### 2.1.3 六种 V1 商品交付形态

- 文件快照 / 版本订阅
- 只读共享
- API / 服务
- 模板查询 lite
- 查询沙箱
- 固定报告 / 结果产品

### 2.1.4 核心字段主轴

- `rights_type`：权利主轴
- `sku_type`：标准可售对象主轴
- `delivery_mode`：交付轴
- `pricing_mode`：计费轴
- `product_type`：目录与展示分类轴

### 2.1.5 Product 与 SKU 事实源

- `Data Product`：目录、审核、搜索、展示事实源
- `Product SKU`：下单、合同、授权、交付、验收、账单、结算事实源

### 2.1.6 API 类 SKU 默认口径

- 默认权利：`access + result_get`
- 模板命名：`API_SUB / API_PPU` 细分模板
- 不再混用：`ACCEPT_API_V1 / REFUND_API_V1`
