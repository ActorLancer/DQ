# 买家控制台实现完成 ✅

## 概述

买家控制台（Buyer Console）已全部实现完成，包含 7 个核心页面，覆盖买家在数据交易平台的完整工作流程。

## 已完成页面

### 1. Dashboard - 仪表盘
**路径**: `/console/buyer`

**功能**:
- 4 个统计卡片：活跃订阅、待审批申请、本月 API 调用、本月支出
- 活跃订阅列表（配额进度条、到期警告）
- 待审批申请列表
- 系统提醒
- 使用趋势图表（占位）

### 2. Subscriptions - 我的订阅
**路径**: `/console/buyer/subscriptions`

**功能**:
- 搜索和筛选（商品名称、供应商、状态）
- 左侧订阅列表（卡片式）
  - 配额使用进度条（颜色区分）
  - 统计信息（今日调用、总调用、剩余天数）
  - 到期警告（< 30 天）
- 右侧订阅详情
  - Subscription ID
  - 商品信息、授权范围、使用情况
  - 链上凭证状态
  - 操作按钮（查看 API Key、查看日志、续订）

### 3. Requests - 我的申请
**路径**: `/console/buyer/requests`

**功能**:
- 搜索和筛选（商品名称、供应商、状态）
- 左侧申请列表（卡片式）
  - 8 种状态标签
  - 使用用途展示
  - 审核备注提示（颜色区分）
- 右侧申请详情
  - Request ID
  - 申请内容（使用用途、预计用量）
  - 状态信息（申请状态、链状态、投影状态）
  - 审核备注
  - 操作按钮（补充材料、前往订阅管理）

### 4. Orders - 订单与账单
**路径**: `/console/buyer/orders`

**功能**:
- 3 个统计卡片：累计支出、待支付、已开发票
- 搜索和筛选（订单号、商品名称、供应商、状态）
- 左侧订单列表（卡片式）
  - 订单类型标签（订阅、一次性、续订）
  - 订单金额和订单号
  - 发票状态
  - 待支付订单快速操作
- 右侧订单详情
  - Order ID
  - 商品信息、金额信息、状态信息
  - 发票信息
  - 操作按钮（立即支付、取消订单、申请发票、下载发票）

### 5. API Keys - API 密钥管理
**路径**: `/console/buyer/api-keys`

**功能**:
- 安全提示卡片（5 条安全建议）
- API Key 列表表格
  - Key 脱敏显示
  - 状态标签（活跃、已禁用、已过期）
  - 调用次数、最近使用、过期时间
  - 操作按钮（轮换、禁用、删除）
- 创建 API Key Modal
  - Key 名称、关联订阅、过期时间
  - IP 白名单（可选）
- 显示新 Key Modal
  - **红色警告**：仅显示一次
  - 完整 Key 显示
  - 复制按钮（带反馈）

### 6. Usage - 使用分析
**路径**: `/console/buyer/usage`

**功能**:
- 筛选器（时间周期、订阅筛选）
- 4 个统计卡片：总调用、成功率、失败次数、平均响应时间
- 图表区域（占位 + 实现说明）
  - 调用趋势图
  - 响应时间分布图
  - 错误分析图
- 订阅使用排行
  - 排名、总调用、成功/失败、成功率、响应时间
  - 配额使用进度条
- 调用分布饼图（占位 + 图例）

### 7. Settings - 设置
**路径**: `/console/buyer/settings`

**功能**:
- 左侧导航（5 个标签页）
- **个人信息**：姓名、职位、邮箱、手机号、头像
- **企业信息**：企业名称、统一社会信用代码、企业类型、注册地址、法定代表人、注册资本、营业执照上传
- **通知设置**：邮件通知（6 项）、短信通知（3 项）、站内通知（1 项）
- **安全设置**：登录密码、双因素认证、登录设备、API 访问控制
- **账单设置**：发票信息、收件信息、自动开票
- 保存成功提示（右上角绿色提示框）

## 如何访问

启动开发服务器：
```bash
cd apps/portal-web
pnpm dev
```

访问地址：
- Dashboard: http://localhost:3001/console/buyer
- 我的订阅: http://localhost:3001/console/buyer/subscriptions
- 我的申请: http://localhost:3001/console/buyer/requests
- 订单账单: http://localhost:3001/console/buyer/orders
- API 密钥: http://localhost:3001/console/buyer/api-keys
- 使用分析: http://localhost:3001/console/buyer/usage
- 设置: http://localhost:3001/console/buyer/settings
