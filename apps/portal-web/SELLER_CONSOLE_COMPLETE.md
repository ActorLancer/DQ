## 🎯 核心功能详解

### 1. 创建商品向导 - ⭐ 完整 6 步流程

这是供应商后台最复杂的功能，完整实现了从基础信息到提交审核的全流程。

#### Step 1: 基础信息
```
✅ 商品名称（必填）
✅ 行业分类（8 个选项）
✅ 数据类型
✅ 商品简介（100-500 字）
✅ 覆盖范围
✅ 更新频率（5 个选项）
✅ 填写提示卡片
```

#### Step 2: Schema 与样例
```
✅ 动态字段定义
   - 字段名称
   - 字段类型（6 种：字符串、数字、布尔、日期、对象、数组）
   - 字段描述
   - 必填/敏感标记
   - 添加/删除字段

✅ 样例数据输入（JSON 格式）
✅ 空状态提示
✅ Schema 定义提示
```

#### Step 3: 质量与合规
```
✅ 数据来源说明（必填）
✅ 权属证明上传（拖拽上传）
✅ 隐私保护策略（必填）
✅ 质量报告上传
✅ 合规材料上传
✅ 禁止使用场景（必填）
✅ 合规要求警告卡片（黄色）
```

#### Step 4: 交付配置
```
✅ 交付方式多选
   - API 接口（实时查询）
   - 文件下载（批量数据）
   - 数据流（实时推送）

✅ API 配置（条件显示）
   - API Endpoint
   - 鉴权方式（API Key、OAuth 2.0、JWT）
   - API 文档上传

✅ 文件配置（条件显示）
   - 文件格式（CSV、JSON、Excel、Parquet）
   - 更新周期

✅ 交付配置提示
```

#### Step 5: 套餐定价
```
✅ 定价模型选择
   - 按量计费
   - 订阅制
   - 混合模式

✅ 动态套餐配置
   - 套餐名称
   - 价格（元）
   - 调用额度
   - 有效期（天）
   - 添加/删除套餐

✅ 默认套餐模板
   - 试用版（免费、100 次、7 天）
   - 标准版（¥999、10000 次、30 天）
   - 企业版（¥9999、100000 次、365 天）

✅ 定价建议卡片
```

#### Step 6: 提交审核
```
✅ 基础信息摘要
✅ Schema 摘要（字段数量、字段列表）
✅ 质量与合规摘要
✅ 交付配置摘要
✅ 套餐定价摘要（卡片展示）
✅ 提交前确认提示（黄色）
✅ 服务协议勾选
```

#### 导航和操作
```
✅ 左侧步骤指示器
   - 6 步流程展示
   - 当前步骤高亮（蓝色）
   - 已完成步骤打勾（绿色）
   - 步骤间连接线

✅ 底部操作按钮
   - 上一步（第一步禁用）
   - 保存草稿
   - 下一步（1-5 步）
   - 提交审核（第 6 步）
```
