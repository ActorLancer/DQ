# 配置键 Schema

- `APP_ENV`: `local|staging|demo`
- `LOG_LEVEL`: `trace|debug|info|warn|error`
- `POSTGRES_PORT`: 整数端口
- `REDIS_PORT`: 整数端口
- `KAFKA_BIND_HOST`: IP 地址或主机名，默认 `127.0.0.1`
- `KAFKA_PORT`: 整数端口，默认 `9092`
- `KAFKA_EXTERNAL_BIND_HOST`: IP 地址或主机名，默认 `0.0.0.0`
- `KAFKA_EXTERNAL_PORT`: 整数端口
- `KAFKA_EXTERNAL_ADVERTISED_HOST`: IP 地址或 DNS；局域网访问时必须是远端可达地址
- `MINIO_API_PORT`: 整数端口
- `OPENSEARCH_HTTP_PORT`: 整数端口
- `KEYCLOAK_PORT`: 整数端口
